Pretty In Pre

The pre element is used to define pre-formatted text. It preserves spaces and line breaks in the text, making it useful for displaying code or text within the whitespace.

Below is a short JavaScript code displayed in a regular tag:

const greeting = 'Hello, World!'; console.log(greeting);

Now, I have the same JavaScript code in a pre tag to show the formatting differences: 

                const greeting = 'Hello, World!';
                console.log(greeting);
